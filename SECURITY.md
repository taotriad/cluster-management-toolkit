# Security Policy

The cluster-management-toolkit developers are committed to rapidly addressing security vulnerabilities
affecting our users and providing clear guidance on the solution, impact, severity and mitigation.

## Reporting a Vulnerability

Please report any security vulnerabilities in this project using the **New draft security advisory**
button on the [Security/Advisories](https://github.com/taotriad/cluster-management-toolkit/security/advisories)
page in our GitHub repository.
